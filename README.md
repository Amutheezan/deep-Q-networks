# deep-Q-networks

Implementation of algorithms from the Q-learning family

## Implemented (and will be implemented):
- Vanilla DQN + Nstep + Double
- Dueling DQN
- Prioritized experience replay + DDQN
- C51 (Categorical DQN)

## Dependencies:
- PyTorch = 0.4.1
- Python  = 3.6
- Gym     = 0.12.5
